# NOTE-TAKING-APP

This is a simple note-taking application with a Nest.js backend and Next.js frontend.

## Setup

1. **Clone the repository:**

2. **  run commands **

   - cd backend
   - npm install
   - npm run start

   
3. **  run commands **

   - cd client
   - npm install
   - npm run dev

4 ** enter url on browser **

   - http://localhost:3001/
